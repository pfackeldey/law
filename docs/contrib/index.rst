**contrib** Packages
====================

.. automodule:: law.contrib

To use on of the following packages in your code, you must import them explicitly, e.g.

.. code-block:: python

   import law

   import law.contrib.git
   import law.contrib.htcondor
   # or
   law.contrib.load("git", "htcondor")
